# Benchmarking-Deep-RL-Algorithms-with-Potential-Based-Reward-Shaping-on-LunarLander
It trains DQN, PPO, and A2C (from Stable-Baselines3) on the discrete LunarLander-v3 environment under 4 reward configurations - none, distance potential, angle potential, and combined - and evaluates sample efficiency and final performance.
