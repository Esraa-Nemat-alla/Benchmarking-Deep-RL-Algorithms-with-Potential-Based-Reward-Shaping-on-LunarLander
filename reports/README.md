# Reports Directory

Generated project outputs are organized here:

- `figures/`: evaluation plots generated by `python evaluate.py`
- `tables/`: CSV summaries generated by `python evaluate.py`
- `demos/`: replay GIFs generated by `python demo.py`
- `RESULTS_INTERPRETATION.md`: written summary generated by `python analyze_results.py`

Recommended files for presentation:

- `figures/comparative_bar_chart.png`
- `tables/results_summary.csv`
- `tables/best_configs.csv`
- `tables/reward_shaping_effects.csv`
- `demos/td3_combined.gif`
